# JeffJump
JeffJump est un jeu basé sur le célèbre DoodleJump ! Il est adapté au monde des fameux et légendaires Tuche. Et au coeur de ce projet, on retrouve la NevadIA, un algorithme génératif qui joue à JeffJump.
